# todo-app
A Todo application
      A simple and efficient task management application that allows users to create, organize, and track their daily tasks.

Overview

      This project is built to demonstrate core frontend development concepts such as DOM manipulation, event handling, and state management. The application provides an intuitive interface for managing tasks in real time.

Features
  Add new tasks
  Delete existing tasks
  Mark tasks as completed
  Dynamic UI updates without page reload
  
Tech Stack
  HTML
  CSS
  JavaScript
  
Purpose
  The goal of this project is to strengthen fundamental programming concepts and build a foundation for more advanced full-stack applications.
